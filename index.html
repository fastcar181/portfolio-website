Project #2: Robot Delivery Program

Description:

Languages: Java

Software and tools: Webots

This was a final team project for an introduction to robotics course (COMP 2801). The purpose of this program was to implement a search algorithm and collision-free movement that would allow the robot to collect green jars and deliver them to the dropoff location, stopping the simulation once two green jars were delivered.

Highlights:

LiDAR: I implemented a simple LiDAR detection system to help prevent collisions.

Camera detection: I utilized the built-in Webots camera to help with the process of identifying green jars and the green dropoff location.

State machines: I planned out state machines for the robot's various states before implementation, ensuring all cases were covered.

Challenges:

Collisions: To create a reliable collision-free movement, I implemented both distance and LiDAR sensors and recorded the numbers for the sensors to gauge the numbers to use in the program.

Project #3: Steam Statistics

Description

Languages: Python

Software and tools: Steam API, Git, GitHub, and Visual Studio Code

This is a super simple Steam-based program that fetches some info on a specific Steam profile, given a Steam API key and Steam ID. I enjoy playing games, so I wanted to dip my toes into creating a program that involves games in one way or another. I'm excited to expand this particular project by implementing a GUI, allowing for a more user friendly interface!

Highlights:

API: This was my first time using an API, but I was really hooked when I first got it working by fetching my Steam profile's display name. This caused me to read the Steam API documentation and have a little fun playing around with this API.

Challenges:

API: Since I'm new to APIs, I was a little confused, but I quickly got the hang of it by simply conducting research (watching videos, reading documentation, etc.).
